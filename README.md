## 📐 Pythagorean Theorem — Chapter Overview

*[Teorema\_di\_Pitagora.pdf](Pdf/Teorema_di_Pitagora.pdf)*

1. **Historical Introduction** — Pythagoras, ancient Greek mathematics, and the origins of the theorem
2. **Statement of the Theorem** — Formal statement, notation, and geometric interpretation
3. **Classical Proofs** — Multiple approaches: geometric rearrangement, algebraic, and visual proofs
4. **Converse of the Theorem** — Conditions sufficient to identify a right triangle
5. **Pythagorean Triples** — Integer solutions, Euclid's formula, and enumeration
6. **Applications** — Distance formula, trigonometry, coordinate geometry, real-world problems
7. **Exercises and Problems** — Progressive exercises from introductory to advanced level

## ⭕ Treatise on the Circle — Chapter Overview

*[Trattato\_del\_Cerchio Vol. 1–6](Pdf/)*

A comprehensive six-volume treatise exploring the geometry of the circle from its foundational definitions through advanced theorems and applications.

| Volume | Topics Covered |
|---|---|
| **[Vol. 1](Pdf/Trattato_del_Cerchio_Vol1.pdf)** | Definitions, basic properties, diameter, radius, circumference and area formulas |
| **[Vol. 2](Pdf/Trattato_del_Cerchio_Vol2.pdf)** | Arcs, chords, central angles, inscribed angles and their relationships |
| **[Vol. 3](Pdf/Trattato_del_Cerchio_Vol3.pdf)** | Tangent lines, secants, tangent–chord and tangent–secant angle theorems |
| **[Vol. 4](Pdf/Trattato_del_Cerchio_Vol4.pdf)** | Circles and polygons — inscribed, circumscribed, and regular polygons |
| **[Vol. 5](Pdf/Trattato_del_Cerchio_Vol5.pdf)** | Power of a point, radical axis, coaxial circle systems |
| **[Vol. 6](Pdf/Trattato_del_Cerchio_Vol6.pdf)** | Advanced theorems, synthesis, and fully solved application problems |
| **[Appendix](Pdf/Trattato_del_Cerchio_Appendice.pdf)** | Supplementary material and reference results complementing Volumes 1–6 |

## ℹ️ About

This repository is a growing open library of mathematics documentation. Topics covered include — but are not limited to:

- **Geometry** — classical theorems, coordinate geometry, trigonometry
- **Algebra** — equations, polynomials, number theory
- **Analysis** — limits, derivatives, integrals
- **Probability & Statistics** — combinatorics, distributions

All documents are freely accessible and downloadable with no restrictions.

## 🚀 How to Access

1. **Browse online** — click any `.pdf` link in the table above to open it directly in GitHub
2. **Download a single file** — navigate to `Pdf/`, click the file, then choose **Raw** → save from your browser
3. **Clone the full repository**:

```bash
git clone https://github.com/OTHack1A/Mathematics.git
```
