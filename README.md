## About

This repository is a growing library of mathematics documentation. Topics covered include (but are not limited to):

- **Geometry** — classical theorems, coordinate geometry, trigonometry
- **Algebra** — equations, polynomials, number theory
- **Analysis** — limits, derivatives, integrals
- **Probability & Statistics** — combinatorics, distributions

All documents are written to be accessible to high-school and early university students.

---

## How to use

1. Browse the `Pdf/` folder for available documents
2. Click any `.pdf` file to open it directly in the GitHub preview
3. Download individual files or clone the entire repository:

```bash
git clone https://github.com/OTHack1A/mathematics.git
```
