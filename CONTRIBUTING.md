# Contributing to Mathematics — Open Documentation

Thank you for your interest in contributing! This repository welcomes new documents, corrections, and improvements from anyone.

---

## How to contribute

### 1. Adding a new PDF document

1. **Fork** this repository
2. Place your PDF inside the `Pdf/` folder
3. Use a clear, descriptive filename in English (e.g. `Binomial_Theorem.pdf`)
4. Update `README.md` to add your document to the contents table
5. Open a **Pull Request** with a short description of what the document covers

### 2. Fixing an existing document

If you spot an error in an existing PDF or in the README:

1. Fork and clone the repository
2. Make your correction
3. Open a Pull Request explaining what was wrong and what you changed

### 3. Suggesting a topic

Open a **GitHub Issue** with the label `topic request` and describe the theorem or concept you'd like to see covered.

---

## File naming convention

| Rule | Example |
|---|---|
| Use underscores instead of spaces | `Teorema_di_Pitagora.pdf` |
| Start with a capital letter | `Quadratic_Formula.pdf` |
| Be descriptive, not generic | `Euler_Identity_Proof.pdf` ✓ — `Math1.pdf` ✗ |

---

## Document quality guidelines

- The document should be **accurate** and **self-contained**
- Include a clear **statement** of the theorem or concept
- Include at least one **proof** or derivation where applicable
- Diagrams and examples are encouraged
- Language: English or Italian (both accepted)

---

## Code of conduct

Be respectful and constructive in all interactions. This is an educational project open to everyone regardless of background or skill level.
